# Add link from ForensicLabs.gr to the Apps site

Target apps site:

```text
https://apps.forensiclabs.gr/
```

Recommended visible text:

```text
Εφαρμογές / Apps
```

or:

```text
F.I.C. Windows Utilities / Apps
```

Use one of these files:

```text
ADD_APPS_LINK_TO_FORENSICLABS_GR.html
MENU_ITEM_APPS_LINK.html
FOOTER_APPS_LINK.html
```

The link is electric blue and underlined:

```text
#005CFF
```

Hover color:

```text
#00A6FF
```

This does not modify the apps site logic. It is only the backlink that should be placed on the existing main site `https://www.forensiclabs.gr/`.


## Direct link to Apps Statistics

Exact relative link requested:

```text
FIC_Windows_Utilities_Site/index.html#statistics
```

Recommended placement in the current main site layout:

```text
Left column → Επισκέπτες box
```

Ready files:

```text
STATISTICS_LINK_RELATIVE_FOR_MAIN_SITE.html
STATISTICS_LINK_ABSOLUTE_FOR_MAIN_SITE.html
VISITORS_BOX_APPS_STATISTICS_LINK.html
MENU_ITEM_APPS_STATISTICS_LINK.html
```

Use the relative version if the apps site folder is uploaded inside the same hosting root.
Use the absolute version if the apps site is deployed as `apps.forensiclabs.gr`.
